# Видео-фоны

Загрузите сюда файлы для использования в качестве фоновых видео.
Поддерживаются форматы .mp4 и .webm (без звука). Если файл отсутствует,
автоматически используется изображение-постер.

Ожидаемые имена файлов:
- hero.mp4 / hero.webm        — главный экран
- about.mp4 / about.webm      — раздел «О студии»
- events.mp4 / events.webm    — раздел «Мероприятия»
